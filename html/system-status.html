故障排除

可能原因和解决方案：
  • Streamlit未启动 - 运行 python -m streamlit run app.py
  • 端口被占用 - 尝试使用其他端口：--server.port 8502
  • 防火墙阻止 - 检查防火墙设置，允许端口8501
  • Python环境问题 - 确保Python 3.8+已安装

解决步骤：
  1. 更新pip：python -m pip install --upgrade pip
  2. 安装依赖：pip install streamlit pandas numpy plotly scikit-learn scipy
  3. 如果网络问题，使用国内镜像：pip install -i https://pypi.tuna.tsinghua.edu.cn/simple streamlit
  4. 创建虚拟环境：python -m venv venv

检查项目：
  • 浏览器兼容性 - 使用Chrome、Firefox或Safari最新版本
  • 缓存问题 - 清除浏览器缓存或使用无痕模式
  • JavaScript支持 - 确保浏览器启用JavaScript
  • 网络连接 - 检查本地网络连接
