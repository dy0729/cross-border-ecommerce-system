## 🚀 快速开始

### 1. 环境准备
```bash
# 安装Python依赖
pip install -r requirements.txt
```

### 2. 启动应用
```bash
# Windows用户
start.bat

# Linux/Mac用户
chmod +x start.sh
./start.sh

# 或者直接使用Python命令
streamlit run app.py --server.port 8501
```

### 3. 访问系统
打开浏览器访问: http://localhost:8501
